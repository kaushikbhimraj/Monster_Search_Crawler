
Instructions to run locally. (MAC-users):

1. Copy the folder on to your Desktop.

2. Open Terminal.

3. cd ./Desktop/Monster_Job_Search/
   chmod +x ./Setup.sh
   ./Setup.sh

4. Enter your system password when ever prompted.

5. At a certain point the installation will pause, copy & paster -> export PATH=/usr/local/bin:$PATH

6. Press control + x, then y and hit ENTER.

9. After process ends, type -> python3 Monster_Main.py

10. Enter the job title, city, state and posted date.

11. A .JSON file is created in the folder.
